# 🧠 Git Command Cheat Sheet

## 📂 Initial Setup
```bash
git config --global user.name "Your Name"
git config --global user.email "you@example.com"
```

## 🔄 Sync Your Repo
```bash
git clone <repo-url>
cd <repo-folder>
```

## 📝 Add & Commit
```bash
git add .
git commit -m "Your message"
git push
```

## 📥 Pull Latest
```bash
git pull
```

## ⛔ Undo Last Commit
```bash
git reset --soft HEAD~1
```
